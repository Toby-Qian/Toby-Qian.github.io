Things I shipped.

Academic Assistant
01 · Claude Code Skill2026 · MIT

Academic Assignment Assistant

A Claude Code skill that turns a professor-issued assignment file into a submittable Word document — without losing the why. Built around a single principle: understand first, answer second.

It ingests PDF / .docx / .doc via a Python extraction pipeline, decodes both explicit requirements (word count, citation format, deadlines) and implicit ones (grading focus, expected theoretical depth, discipline conventions), then routes by field into separate STEM or humanities playbooks. The output is two (or more) independently complete versions — written from different theoretical frames or technical routes — so the student picks the angle instead of accepting a single draft.

Citations are generated in GB/T 7714 · APA · MLA · Chicago · IEEE, aligned one-to-one with the argument; anything it can't verify is marked 〔to be verified〕 instead of fabricated. The final python-docx build ships a cover page, auto TOC, versioned body sections and a reference list in one file.

02 · Web / Canvas Tool2026 · MIT

Paper Figure Studio

A pure front-end web app that turns an ordinary photo into a figure that looks like it belongs in a paper. No install, no upload — everything renders locally on Canvas 2D, nothing ever leaves the browser.

One click switches the whole layout between Nature · Science · Cell · IEEE · Elsevier house styles — serif stack, accent colour, caption metrics and label weight all swap together. A 6-preset multi-panel grid lets you drop in several images at once; each image can be dragged and wheel-scaled inside its cell, with (a)(b)(c) labels auto-assigned and cleaned up on layout change.

Annotations — arrows, scale bars, text, rectangles, circles — live on a unified selection model, so the same Aa panel drives both captions and labels. Built with vanilla HTML · CSS · JS, a translucent mobile drawer (backdrop-filter + peek mode), and a landing page that doubles as a worked example.
