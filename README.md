# Personal Website

一个可直接部署到 GitHub Pages 的个人网站模板。

## 文件说明

- `index.html`: 页面结构
- `style.css`: 页面样式
- `script.js`: 前端交互

## 本地预览

直接双击 `index.html` 即可在浏览器打开。

## 部署到 GitHub Pages

1. 新建一个 GitHub 仓库，仓库名使用 `Toby-Qian.github.io`
2. 把当前目录里的文件上传到仓库根目录
3. 在仓库里打开 `Settings -> Pages`
4. 选择 `Deploy from a branch`
5. 选择 `main` 分支和 `/root`
6. 保存后等待几分钟
7. 访问 `https://toby-qian.github.io/`
